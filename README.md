# Pac-Man Game

A terminal-based Pac-Man game written in C.

## How to Build

```bash
cmake -S . -B build
cmake --build build
```

## How to Run

```bash
./build/bin/pacman
```

On Windows:
```
./build/bin/pacman.exe
```

## Controls

- `W` - Move up
- `A` - Move left
- `S` - Move down
- `D` - Move right
- `R` or `Space` - Restart
- `Q` - Quit

## How to Play

Move Pac-Man around the maze and eat all the dots while avoiding the ghosts. You have 3 lives.
